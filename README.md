# 🏠 Smart House Price Predictor

🔗 **Live App:** https://smart-house-price-predictor-lrqve5sc.streamlit.app/

---

## 📌 Project Overview

Smart House Price Predictor is a Machine Learning web application that estimates the price of a house based on key property features.
It provides a simple and interactive interface where users can input house details and instantly get a predicted price.

---

## 🚀 Key Features

* 📊 Real-time house price prediction
* 🧠 Powered by Machine Learning (Linear Regression)
* 🎯 User-friendly interface built with Streamlit
* ⚡ Fast and interactive predictions
* 📱 Clean and responsive UI

---

## 🧾 Input Parameters

The prediction is based on the following features:

* Square Footage
* Number of Bedrooms
* Number of Bathrooms
* Year Built
* Lot Size
* Garage Size
* Neighborhood Quality

---

## 🤖 Model Details

* Algorithm: Linear Regression
* Preprocessing: Standard Scaling
* Output: Predicted House Price

The model is trained on structured housing data and designed to provide accurate and consistent predictions.

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn

---

## 📊 How It Works

User inputs property details → data is processed → model predicts price → result is displayed instantly on the screen.
